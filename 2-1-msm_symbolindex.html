  • $\RNr{n}$ i

    • Consists of the collection of all $n$-tuples $(x_1,\dots ,x_n)$ of numbers.


    The set $\RNr{n}$ Definition

    For a positive integer $n$, the space $\RNr{n}$ is the set of all $n$-tuples of numbers; in symbols

    $$\RNr{n}\ :=\ \Set{(x_1,\dots ,x_n)\st x_1,\dots ,x_n\in \RNr{} }$$

    • Explanation

      • Read this expression as follows:

      R n is defined to be the set of all n-tuples $(x_1,\dots ,x_n)$ such that $x_1,\dots, x_n$ are in $\RNr{}$.

    • Explanation

      • Thus, each $n$-tuple forms a single element in $\RNr{n}$. The collection of all such $n$-tuples forms the space $\RNr{n}$.

    Length of an Arrow

    Given points $P(x_1,\dots ,x_n)$ and $Q(y_1,\dots ,y_n)$ in $\RNr{n}$, the length of the arrow $\Arrow{P}{Q}$ from $P$ to $Q$ is

    $$\Length{\Arrow{P}{Q} }\ :=\ \Dstnc{P}{Q}\ =\ \sqrt{(y_1-x_1)^2+\cdots +(y_n-x_n)^2}.$$

    • Illustration
      • Illustration: Length of an Arrow

      The length of the arrow joining $P(x_1,\dots ,x_n)$ to $Q(y_1,\dots ,y_n)$ is given by the formula

      $$\Abs{ \overset{\longrightarrow}{PQ} }\ :=\ \text{dist}(P,Q)\ =\ \sqrt{(y_1-x_1)^2+\cdots +(y_n-x_n)^2}.$$

      This means that the length of an arrow is defined to be the distance between its end points.

      1. going from $P$, $x_1$ units in the first coordinate direction

      2. from there $x_2$ units in the second coordinate direction,

      3. etc. .... until

      4. from the last found position $x_n$ units in the $n$-th coordinate direction.

  • $\sphericalangle(\Vect{x},\Vect{y})$ i

    • denotes the angles between two vectors $\Vect{x}$ and $\Vect{y}$.


    Geometric Properties of Norm and Dot ProductProposition

    For vectors $\Vect{x}$ and $\Vect{y}$ in $\RNr{n}$, the following hold:

    1. Relationship between dot product and norm

      $\DotPr{ \Vect{x} }{ \Vect{x} } = | \Vect{x} |^2$


    2. Orthogonality criterion

      $\DotPr{\Vect{x}}{\Vect{y}} = 0$ if and only if $\Vect{x}$ is perpendicular to $\Vect{y}$.


    3. Angle between two vectors

      $\sphericalangle(\Vect{x},\Vect{y})$ denotes the angle between the vectors $\Vect{x}$ and $\Vect{y}$.

      $\DotPr{\Vect{x}}{\Vect{y}} = \Abs{ \Vect{x} } \Abs{ \Vect{y} }\cdot \cos \sphericalangle(\Vect{x},\Vect{y})$ , provided $\Vect{x}$ and $\Vect{y}$ have positive length.


    4. Triangle inequality

      $\Abs{\Vect{x} + \Vect{y}} \leq \Abs{\Vect{x}} + \Abs{\Vect{y}}$


    5. Cauchy-Schwarz inequality

      $| \DotPr{\Vect{x}}{\Vect{y}} | \leq | \Vect{x} | | \Vect{y} |$.

  • $\pr_L(\Vect{x})$ i

    Orthogonal Projection of a Vector on a LineProposition

    Let $\Vect{x}$ be an arbitrary vector of $\RNr{n}$, and let $L$ be the line through the origin in the direction of a nonzero vector $\Vect{y}$. Then the orthogonal projection of $\Vect{x}$ onto $L$ is the vector

    $$\pr_L(\Vect{x}) = \dfrac{ \DotPr{\Vect{x}}{\Vect{y}} }{ \DotPr{\Vect{y}}{\Vect{y}} } \cdot \Vect{y}$$

    Moreover, $\Vect{y}$ is perpendicular to $\Vect{x} - \pr_{L}(\Vect{x})$.

          It is very likely that you have already come in contact with the concept of a function. For example, within the context of calculus, you might have gotten used thinking of a function as a curve in a 2D-coordinate system like this one:

          For most of our present purposes the graph is not the appropriate object to associate to a function. Therefore it is necessary to take the concept of a function in its original form and then learn how to use it to transforms space.

          A set is any collection $U$ of objects, even objects of your thought or imagination are allowed. The objects in $U$ are called the elements of $U$. We write $u\in U$ to say that $u$ is an element of $U$.

          Setting up a function begins with selecting two sets

          • A set $X$, called the domain of the function

          • A set $Y$, called the target of the function

          A function $f$ from $X$ to $Y$, denoted $f\from X\to Y$, associates to each element $x$ in $X$ some element of $Y$. This element must be unique and is denoted $f(x)$. Names for it include: the ‘value’ of $f$ at $x$, the ‘function value’ of $f$ at $x$, or the ‘image of $x$ under $f$ ’.

          Functions and equations   We just learned that a function $f\from X\to Y$ can transform several distinct elements of $X$ into the same element $y\in Y$. Therefore it makes sense to ask: What is the total collection of elements of $X$ which get sent to a given $y$ in $Y$? or, in symbols, what are the solutions of the equation below?

          $$f(x) = y$$

          Thus functions provide a general tool to formulate equations. The solutions of the equation above form the ‘preimage of $y$ under $f$’. It is denoted $f^{-1}(y)$ and consists of all those $x$ in $X$ which satisfy the equation $f(x)=y$.

          There remains the question: How do we solve such equations? In general, this task can be quite challenging. At this point we can indicate the following:

          • If $f$ is a linear function, we will see that the equation above is equivalent to a system of linear equations, and so we know how to solve them.

          • If $f$ is differentiable, we can feed our knowledge of solving linear equations into the framework of differential calculus and obtain qualitative answers. These answers include the ‘inverse function theorem’ or its corollary the ‘implicit function theorem’.

          • If $f$ is a polynomial, i.e. a relatively ‘simple’ function, efforts to describe all of its solutions continue to provide ongoing research challenges, and is the primary goal of the subject of Algebraic Geometry.

            • Cross Product and Quaternion Number System

            There is a notational convention, used by engineers and physicists, which may make the formula for the cross product a little easier to remember. Let us name the standard coordinate vectors of $\RNr{3}$ as.

            $\Vect{i}:=(1,0,0)$,   $\Vect{j}:= (0,1,0)$,   $\Vect{k}:=(0,0,1)$

            Let us be clear: Strictly speaking, the proposed determinant does not exist, as the entries in the first column are not numbers. However, nothing can stop us from applying the formula for evaluating $(3,3)$-determinants to the symbols appearing in the proposed matrix. Magically the formula for the cross product results. Only now we remember that $\Vect{i}$, $\Vect{j}$, and $\Vect{k}$ are vectors in $\RNr{3}$.

            Now a vector $\Vect{z}=(a,b,c)$ can be written as $\Vect{z} = a \Vect{i} + b \Vect{j} + c \Vect{k}$. The formula for the cross product results from symbolically evaluating the determinant below along the first column. Here $\Vect{x} = (x_1,y_1,z_1)$ and $\Vect{y} = (y_1,y_2,y_3)$.

            $$ \aligned \CrssPr{ \Vect{x} }{ \Vect{y} }\ &=\ \det \left[ \begin{array}{ccc} \Vect{i} & x_1 & y_1 \\ \Vect{j} & x_2 & y_2 \\ \Vect{k} & x_3 & y_3 \end{array} \right] \\ &=\ \det \left[ \begin{array}{cc} x_2 & y_2 \\ x_3 & y_3 \end{array} \right]\Vect{i}\ +\ (-1)\det \left[ \begin{array}{cc} x_1 & y_1 \\ x_3 & y_3 \end{array} \right] \Vect{j}\ +\ \det \left[ \begin{array}{cc} x_1 & y_1 \\ x_2 & y_2 \end{array} \right] \Vect{k} \\ &=\ (x_2y_3-x_3y_2 , -(x_1y_3-x_3y_1) , x_1y_2 - x_2y_1) \endaligned $$

            Recalling that, for a square matrix $\Mtrx{A}$, $\det(\Mtrx{A}) = \det(\Mtrx{A}^T)$, we can alternatively express the above formula as follows:

            $$ \aligned \CrssPr{ \Vect{x} }{ \Vect{y} }\ &=\ \det \left[ \begin{array}{ccc} \Vect{i} & \Vect{j} & \Vect{k} \\ x_1 & x_2 & x_3 \\ y_1 & y_2 & y_3 \end{array} \right] \\ &=\ \det \left[ \begin{array}{cc} x_2 & y_2 \\ x_3 & y_3 \end{array} \right]\Vect{i}\ +\ (-1)\det \left[ \begin{array}{cc} x_1 & y_1 \\ x_3 & y_3 \end{array} \right] \Vect{j}\ +\ \det \left[ \begin{array}{cc} x_1 & y_1 \\ x_2 & y_2 \end{array} \right] \Vect{k} \\ &=\ (x_2y_3-x_3y_2 , -(x_1y_3-x_3y_1) , x_1y_2 - x_2y_1) \endaligned $$

            1. Write an element $(a,b,c,d)$ of $\RNr{4}$ as $a + b \Vect{i} +c \Vect{j} + d \Vect{k}$.

            • Cross Products in Arbitrary Dimensions?

            The operations ‘dot product’ and ‘determinant’ are defined for all dimensions $n\geq 1$. Analogously we would expect a definition of the cross product in dimensions $n\geq 1$. There are two interpretations

            • We look for an alternating and multilinear operation which assigns to an ordered $(n-1)$-tuple of vectors $(\Vect{b}_1,\dots ,\Vect{b}_{n-1})$ in $\RNr{n}$ a vector $\text{Cross}(\Vect{b}_1,\dots ,\Vect{b}_{n-1})$ in $\RNr{n}$ such that

              $\det[\Vect{b}_1\ \ \cdots\ \ \Vect{b}_{n-1}\ \ \text{Cross}(\Vect{b}_1,\dots ,\Vect{b}_{n-1})]$ $ > $ $0$

              $(\Vect{b}_1,\dots ,\Vect{b}_{n-1})$ is linearly independent. Such an operation exists, but is usually not referred to as a ‘cross product’.

            • We look for an alternating bilinear operation on ordered pairs of vectors $(\Vect{u},\Vect{v})$ such that $\CrssPr{ \Vect{u} }{ \Vect{v} }\neq \Vect{0}$ whenever $\Vect{u}$ and $\Vect{v}$ are not parallel, and such that $\CrssPr{ \Vect{u} }{ \Vect{v} }$ is perpendicular to both $\Vect{u}$ and $\Vect{v}$.

              For deeper reasons, such an operation is only possible if $n=3$ or $n=7$.

            Advanced information   We mention two results on the (non-)existence of cross products of pairs. First a now classical result from algebraic topology:

            Theorem   Suppose $\RNr{n}$ admits a cross product operation satisfying

            • If $\Vect{x}$ and $\Vect{y}$ are not parallel, then $\CrssPr{ \Vect{x} }{ \Vect{y} }\neq 0$.

            • For all $\Vect{x},\Vect{y}\in \RNr{n}$, $\DotPr{ (\CrssPr{ \Vect{x} }{ \Vect{y} }) }{ \Vect{x} }\neq 0$ and $\DotPr{ (\CrssPr{ \Vect{x} }{ \Vect{y} }) }{ \Vect{y} }\neq 0$

            • $\CrssPr{-}{-}\from \RNr{n}\times \RNr{n} \longrightarrow \RNr{n}$ is a continuous function

            Then $n=3$ or $n=7$.

            If we add to the list of assumptions above the requirement that the cross product be bilinear, then the conclusion of the theorem can be proven by elementary algebraic means. This is a classical theorem of Frobenius.

                                  represents $L$ in the sense that, for every $\Vect{x}$ in $V$,

                                  $L(\Vect{x})_{\EuScript{B}}$$=$$A_{\EuScript{B}\EuScript{A}}\cdot \Vect{x}_{\EuScript{A}}$

                                  • Matrix Representation of a Linear Map - Explanation

                                  Let's take a careful look at how to work with this proposition.

                                  First of all, recall that $\Vect{x}_{\EuScript{A}} = (x_1,\dots ,x_n)$ is an $n$-tuple of numbers which is defined by the identity

                                  $\Vect{x}$$=$$x_1 \Vect{a}_1 + \cdots + x_n \Vect{a}_n$

                                  Matrix gives linear map   If $\Mtrx{A}=[a_{ij}]$ is a matrix of size $(m,n)$, then the matrix product

                                  $$ \left[ \begin{array}{ccc} a_{11} & \cdots & a_{1n} \\ \vdots & \ddots & \vdots \\ a_{m1} & \cdots & a_{mn} \end{array} \right]\, \left[ \begin{array}{c} x_1 \\ \vdots \\ x_n \end{array} \right] = \left[ \begin{array}{c} y_1 \\ \vdots \\ y_m \end{array} \right] $$

                                  is defined, and the resulting vector $\Vect{y}=(y_1,\dots ,y_m)$ is the coordinate vector of $L(\Vect{x})$ with respect to the basis $\EuScript{B}$. In symbols:

                                  $L(\Vect{x})_{\EuScript{B}}$$=$$y_1\cdot \Vect{b}_1 + \cdots y_m\Vect{b}_m$

                                  So this is how an $(m,n)$-matrix determines a linear map $V\to W$ with respect to the bases $\EuScript{A}$ and $\EuScript{B}$. – Keeping the same matrix, but using different bases for $V$ and $W$ will, in general, produce a different linear map.

                                  Linear map gives matrix   Now let's start with a given linear map $L\from V\to W$. We know that $L$ is completely determined by the vectors $L(\Vect{a}_1),\ \dots\ ,L(\Vect{a}_n)$ in $V$. Now, each of these vectors can be expressed uniquely as a linear combination of $\EuScript{B}$. So, if

                                  $L(\Vect{a}_1)$$=$$a_{11}\Vect{b}_1 + \cdots + a_{1m} \Vect{b}_m$

                                  then $L(\Vect{a}_1)_{\EuScript{B}} = (a_{11},\dots ,a_{1m})$ is a vector with $m$ entries. It forms the first column of the matrix $\Mtrx{A}_{\EuScript{B}\EuScript{A}}$ which represents $L$.

                                  Similarly, if

                                  $L(\Vect{a}_2)$$=$$a_{21}\Vect{b}_1 + \cdots + a_{2m} \Vect{b}_m$

                                  then $L(\Vect{a}_2)_{\EuScript{B}} = (a_{21},\dots ,a_{2m})$ is a vector with $m$ entries. It forms the second column of the matrix $\Mtrx{A}_{\EuScript{B}\EuScript{A}}$ which represents $L$; etc.

                                  Note that keeping the same linear map $L$, but using different bases for $V$ and $W$ will, in general, yield a different representing matrix for $L$.

                                  Emphasis   Note that both constructions which translate between linear maps and matrices depend crucially on the choice of an ordered basis $\EuScript{A}$ for $V$ and an ordered basis $\EuScript{B}$ for $W$. The subscripts in $\Mtrx{A}_{\EuScript{B}\EuScript{A}}$ are to remind us that this particular matrix represents the given linear map with respect to the bases $\EuScript{A}$ of $V$ and $\EuScript{B}$ of $W$.

                                      • Linear map from matrix

                                        • For $\Vect{x}$ and $\Vect{y}$ in $V$, we have $(\Vect{x}+\Vect{y})_{\EuScript{A}} = \Vect{x}_{\EuScript{A}} + \Vect{y}_{\EuScript{A}}$. Therefore,

                                        $L(\Vect{x}+\Vect{y})_{\EuScript{B}}$$=$$A_{\EuScript{B}\EuScript{A}}\cdot (\Vect{x} + \Vect{y})_{\EuScript{A}}$
                                        $=$$A_{\EuScript{B}\EuScript{A}}\cdot (\Vect{x}_{\EuScript{A}} + \Vect{y}_{\EuScript{A}})$
                                        $= $

                                        Here we use the distributivity law of matrix multiplication

                                        		$\Mtrx{A}_{\EuScript{B}\EuScript{A}} \Vect{x}_{\EuScript{A}}\ +\ \Mtrx{A}_{\EuScript{B}\EuScript{A}} \Vect{y}_{\EuScript{A}}$
                                        $=$$L(\Vect{x})_{\EuScript{B}} + L(\Vect{y})_{\EuScript{B}}$

                                        Therefore $L(\Vect{x}+\Vect{y}) = L(\Vect{x}) + L(\Vect{y}))$

                                        Next we show that $L$ commutes with scalar multiplication:

                                        $L(t \Vect{x})_{\EuScript{B}}$$=$$A_{\EuScript{B}\EuScript{A}} (t \Vect{x})_{\EuScript{A}}$
                                        $= $

                                        This is so because $t(x_1 \Vect{a}_1 +\cdots + x_n \Vect{a}_n) = (tx_1)\Vect{a}_1 + \cdots + (tx_n) \Vect{a}_n$.

                                        		$\Mtrx{A}_{\EuScript{B}\EuScript{A}}\cdot \left( t(\Vect{x}_{\EuScript{A}})\right)$
                                        $= $

                                        ... because scalar multiplication commutes with matrix multiplication.

                                        		$t\left( \Mtrx{A}_{\EuScript{B}\EuScript{A}} \Vect{x}_{\EuScript{A}} \right)$
                                        $=$$t\left( L(\Vect{x})_{\EuScript{B}}\right)$
                                        $=$$\left( t L(\Vect{x}) \right)_{\EuScript{B}}$

                                        This means that $L(t \Vect{x}) = t L(\Vect{x})$, as required.

                                      • Representing a linear map by a matrix

                                        • From the first part of the theorem we know already that the matrix $A_{\EuScript{B}\EuScript{A}}$ determines a linear transformation $M\from V\to W$. Moreover,

                                        $M(\Vect{a}_j)_{\EuScript{B}}$$=$$\Mtrx{A}_{\EuScript{B}\EuScript{A}} \Vect{a}_j = L(\Vect{a}_j)_{\EuScript{B}}$

                                        Consequently, $M=L$, implying that $\Mtrx{A}_{\EuScript{B}\EuScript{A}}$ represents $L$. It remains to show that $\Mtrx{A}_{\EuScript{B}\EuScript{A}}$ is the only matrix with this property. So suppose $\Mtrx{S}$ is an $(m,n)$-matrix which also satisfies

                                        $L(\Vect{x})_{\EuScript{B}}$$=$$S\cdot \Vect{x}_{\EuScript{A}}$

                                        for every vector $\Vect{x}\in V$. Then we find

                                        $$\left( \Vect{A}_{\EuScript{B}\EuScript{A}} - S\right) \Vect{x}_{\EuScript{A}} = \Mtrx{A}_{\EuScript{B}\EuScript{A}}\Vect{x}_{\EuScript{A}} - S \Vect{x}_{\EuScript{A}} = \Vect{0}_{\EuScript{B}}$$

                                        Now $(\Vect{a}_j)_{\EuScript{A}} = (0,\dots , 0,1,0, \dots ,0)$, the "1" sitting in position $j$. Further, multiplying any matrix $\Mtrx{U}$ by $(0,...,0,1,0,...,0)^T$ on the right yields precisely the $j$-th column vector of $\Mtrx{U}$. This implies that each column of $\left( \Mtrx{A}_{\EuScript{B}\EuScript{A}} - S \right)$ is $\Vect{0}$ and, therefore, the two matrices are equal. So $\Mtrx{A}_{\EuScript{B}\EuScript{A}}$ is the only matrix which represents $L$, and the proof of the theorem is complete.
