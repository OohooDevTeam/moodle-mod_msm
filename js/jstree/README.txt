--------------------------------------------------------------------------------
This jQuery plugin is used in display created units in XML hierarchy panel.
The jTreeview was not used due to its inability to drag and drop each leaves
into different parent nodes.  This plugin lets the user change the structure of the 
current tree by dragging and dropping the tree leaves and nodes.
